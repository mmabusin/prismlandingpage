## Shared design system

- **Masthead:** prism logo mark + `The Prism.` wordmark, high up, with a per-email tag on the right.
- **Accent:** `#2563EB` blue rule + eyebrow (E7 uses green `#16a34a` to mark the access moment).
- **Footer:** wordmark + "You're receiving this…" + unsubscribe. No `— The Prism` sign-off (the logo/footer carry the brand).
- Table-based layout, inline styles, Arial/Helvetica fallbacks — web fonts load only where supported (Apple Mail/iOS).

## Before sending — two things to wire up

1. **Logo image must be live.** Every email references `https://theprismai.com/prism-mark.png` (email clients need an absolute URL; inline SVG doesn't render in Outlook/Gmail). That file is [`../prism-mark.png`](../prism-mark.png) and goes live with the deploy. Until then, inboxes show the alt-text "The Prism".
2. **Placeholders to replace on import:**
   - `{{unsubscribe}}` (footer, all emails) → Loops' unsubscribe merge tag (or let Loops append its own).
   - `{{app_url}}` (E7 button) → the live app URL.

## Caveats

- **Outlook (Word engine)** ignores `border-radius` (chips/cards render square) and is imperfect on `inline-block` chips — everything still shows and reads fine, just less rounded. Send yourself a test through Loops and check Gmail + Outlook + Apple Mail before the first real send.
- The 🇫🇷-style flag in E4 is rendered as `FRA` text for client consistency.
